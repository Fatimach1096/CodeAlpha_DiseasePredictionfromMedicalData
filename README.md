#  Disease Prediction from Medical Data

A Machine Learning project developed as part of the **CodeAlpha Internship**. This project predicts the presence of heart disease using clinical data and compares the performance of multiple machine learning algorithms.

##  Project Overview

The goal of this project is to build and evaluate classification models that can predict whether a patient has heart disease based on medical attributes. Different machine learning models are trained and compared to identify the best-performing algorithm.

##  Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature scaling
- Train-test split
- Multiple Machine Learning models
- Performance comparison
- Confusion Matrix
- Classification Report
- Accuracy Evaluation
